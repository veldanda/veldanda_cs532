1. To compile: gcc -Wall -pthread hw4.c -o hw4

2. Run 'hw4' executable file.
   ./hw4 <No. of jobs>
   Ex: ./hw4 2

3. Enter following commands:
    submit <new executable path> <run time>
    New processes are created

    Showjobs
	It will list the processes that are running.

    submithistory
	It will list all the processes with status.
